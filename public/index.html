El Polinomio de Newton es una forma de interpolación que permite construir un polinomio que pasa exactamente por un conjunto dado de puntos conocidos. La interpolación polinómica es esencial cuando se desea estimar valores intermedios entre los puntos de datos existentes o cuando se requiere una representación continua de datos discretos.

Entrada de Datos: El usuario ingresa un conjunto de puntos conocidos.
Cálculo del Polinomio de Newton: Diferencias Divididas: La aplicación utiliza las diferencias divididas para calcular los coeficientes del polinomio.
Evaluación del Polinomio: Estimación del Valor de \( y \): Utilizando el polinomio construido, la aplicación evalúa el valor de \( y \) en el punto \( x \) especificado por el usuario.
Visualización Gráfica: Gráfica del Polinomio: La aplicación genera una gráfica que muestra el polinomio de Newton junto con los puntos de datos originales.
